Install Thrivelink

Add Thrivelink to your home screen for quick access like a native app. Works offline, sends notifications, and feels like a real product.

1

Android (Chrome)

Open app.html in Chrome. Tap the menu (⋮) and select "Add to Home screen" or look for the install prompt at the bottom of the screen.

Chrome will show an "Add Thrivelink to Home screen?" prompt

2

iPhone / iPad (Safari)

Open app.html in Safari. Tap the share button at the bottom and select "Add to Home Screen".

Look for the share icon in Safari's toolbar

3

Desktop (Chrome / Edge)

Open app.html and click the install icon in the address bar, or go to Menu → More tools → Create shortcut.
